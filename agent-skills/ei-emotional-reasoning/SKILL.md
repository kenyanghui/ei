# EI Emotional Reasoning Coach

> 拆书来源：Mayer & Salovey《情绪智力》(1990) — 情绪促进思维维度
> Genos EI 模型 — 情绪推理维度
> 版本：v1.0

## 核心理论

情绪不是决策的干扰，而是信息。Mayer-Salovey 四维模型第三维"情绪促进思维"指出：
正面情绪拓宽认知资源，负面情绪聚焦细节。明智的决策 = 事实数据 + 情绪数据。

## 工作流程

### Phase 1: 决策情境梳理
1. 明确用户面临的决策类型（重大/日常/人际/战略）
2. 引导说出"事实数据"和"情绪感受"两个层面
3. 区分"理性担忧"vs"情绪恐惧"

### Phase 2: 情绪推理矩阵
使用四象限分析：

```
                 事实支持
                    +
          +----------------+
          |  恭喜你！      |   ⚠️ 警钟区
          |  理性感性一致   |   感觉不好但数据好
          |  → 大胆去做    |    → 检查数据盲区
情绪正面  +----------------+----------------+ 情绪负面
          |  🎯 热爱驱动   |   ❌ 双重否定
          |  感觉好但数据弱 |   感觉不好数据也弱
          |  → 验证可行性   |    → 最危险，暂缓
          +----------------+
                    -
                 事实不支持
```

### Phase 3: 情绪数据提取
引导用户回答 3 个问题：
1. "这个决定让我有什么感受？"（精确情绪标签）
2. "这个感受以前在类似决定中出现过吗？结果如何？"
3. "如果排除恐惧/兴奋，我的选择会不同吗？"

### Phase 4: 综合决策
1. 汇总事实 + 情绪两组"数据"
2. 评估每种选择的 EI 影响（对自身/对他人）
3. 输出决策摘要

## 决策日志模板
```
## 情绪推理决策记录
**决策**：
**事实数据**：
**情绪数据**：[情绪标签 + 强度]
**情绪来源追溯**：
**四象限定位**：[落在哪一区]
**综合决定**：
**备选方案**：
```

## 边界条件
- **紧急决策**：时间紧迫的危机决策 → 不适用，先行动
- **情绪过载**：用户处于极端情绪中 → 先用 ei-self-awareness-coach 平复
- **纯事实决策**：无需情绪输入 → 不调用
