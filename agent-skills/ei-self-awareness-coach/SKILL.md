# EI Self-Awareness Coach

> 拆书来源：丹尼尔·戈尔曼《情商》(1995) — 自我觉察维度  
> 理论根基：Mayer & Salovey 四维能力模型 — 情绪感知维度  
> 版本：v1.0

## 核心方法论

自我觉察是 EI 的基石。没有觉察就没有管理。本 skill 通过"标记 → 溯源 → 模式识别 → 日志化"四步，帮助用户从混沌情绪中获得清晰认知。

## 工作流程

### Phase 1: 情绪标记（Emotional Labeling）

1. 引导用户描述当前情绪感受
2. 帮助用户用精确词汇命名情绪（避免"不好/烦躁"等笼统词）
3. 使用三级精度递进：
   - 一级：喜/怒/哀/惧/厌/惊（基础六类）
   - 二级：在上述基础上细分（如怒 → 不满/恼怒/愤慨/暴怒）
   - 三级：定位复合情绪（如"既失望又解脱"）

**输出**：一个精确的情绪标签 + 强度评分（1-10）

### Phase 2: 溯源分析（Trigger Tracing）

1. 引导用户追溯情绪触发事件
2. 区分"外部事件"vs"内部解读"
3. 完成情绪逻辑链：

```
事件（What happened）
  → 身体反应（What I felt physically）
    → 想法/解读（What I told myself）
      → 情绪（What I felt）
        → 行为冲动（What I wanted to do）
          → 实际行为（What I actually did）
```

**输出**：完整的情绪逻辑链

### Phase 3: 模式识别（Pattern Mining）

1. 询问"这种感受以前出现过吗？"
2. 帮助用户识别重复出现的情绪模式
3. 区分"情境性反应"vs"习惯性反应"
4. 标记高价值模式（建议记录）

**输出**：1-3 个可识别的情绪模式 + 模式标签

### Phase 4: 日志化（Journaling）

1. 生成结构化的情绪日志条目
2. 格式：

```markdown
## 情绪日志
**日期**：YYYY-MM-DD
**触发事件**：
**精确情绪标签**：[情绪名]（强度：X/10）
**身体信号**：
**内在解读**：
**行为冲动 vs 实际行为**：
**模式识别**：[是否见过类似情况]
**洞察**：
```

3. 鼓励用户持续记录（"每天 3 分钟"）

### Phase 5: 行动计划

1. 基于模式识别，建议 1 个可执行的小行动
2. 行动必须具体、微小、可今天做
3. 示例："明天会议前深呼吸3次"而非"控制好情绪"

## 边界条件（何时不适用）

- **急性危机**：用户处于严重心理危机/自伤倾向 → 不执行，引导寻求专业帮助
- **外部归因固着**：用户持续指责外部（"都是XX害的"） → 先引导"即使别人有错，你的感受也是真实的"
- **需要临床诊断**：暗示可能有抑郁/焦虑障碍 → 建议专业评估，本 skill 不能替代治疗
- **纯理性分析需求**：用户想要的是逻辑分析而非情绪觉察 → 转给其他技能

## 用户确认检查点

- Phase 2 开始前确认："这个事件适合深入追溯吗？"
- Phase 4 结束时询问："这个日志条目要存下来还是仅本次？"
