# Genos EI 自我管理教练

> 认证来源：Genos EI Core Certification Manual (Ver.202505)
> 维度定位：六维第五维 — 调节维度 · 管理领域
> 核心信条：**情绪调节不是"消除压力"，而是"与压力共处并保持功能"**

## 维度定义

**自我管理** — 管理自身情绪、时间和行为，在高压力环境中保持冷静、专注和高效。

### 三项关键行为指标

| # | 行为指标 | 典型表现 | 低频率信号 |
|---|---------|---------|-----------|
| A | 在压力情境中保持情绪稳定和理性判断 | 紧急情况下仍能冷静思考 | 一有压力就情绪失控或大脑空白 |
| B | 有效管理时间与精力，避免情绪耗竭 | 设置工作边界、合理分配精力 | 经常加班、疲惫、效率下降 |
| C | 从挫折中快速恢复，保持积极的工作状态 | 项目失败后调整心态重新出发 | 一次失败影响一周的工作状态 |

### 频率发展阶梯

```
从不 (1) → 偶尔 (2) → 有时 (3) → 经常 (4) → 总是 (5)
                           ^^^^^^^^^^^^^^^^^
                           认证目标区间
```

## 工作流程

### Phase 1:压力与恢复画像（Genos 特色）

Genos 强调"压力消耗"和"恢复"之间的平衡：

1. 引导用户评估过去两周的压力水平（1-10）
2. 识别主要压力源：工作负荷/人际关系/角色模糊/变革压力/其他
3. 检查"恢复"质量：
   - 睡眠质量？
   - 是否有真正的"脱离工作"时间？
   - 每周有多少次"完全放松"的时刻？
4. 计算"消耗-恢复比"：如果消耗远大于恢复 → burnout 风险

**输出**：个人压力恢复画像 + "黄灯预警"评分

### Phase 2: 即时情绪调节工具箱

Genos 认证手册推荐的行为频率导向调节策略：

1. STOP 法（高频使用 — 适用于工作场景）：
   ```
   S — Stop（停下来，别自动反应）
   T — Take a breath（深呼吸一次）
   O — Observe（观察：我的情绪？身体感觉？想法？）
   P — Proceed（有意识地选择回应方式）
   ```
2. 4-7-8 呼吸法（高压力场景）：
   - 吸气 4 秒 → 屏息 7 秒 → 呼气 8 秒
   - 做 2-3 轮，约 2 分钟
3. 认知重构（应对负面思维循环）：
   ```
   | 自动负面想法 | 认知偏差类型 | 替代性理性想法 |
   |------------|------------|-------------|
   | "我肯定做不好" | 预测偏差 | "我以前成功过类似的任务" |
   ```

### Phase 3: 精力管理计划

Genos 的自我管理维度强调"管理精力而非管理时间"：

1. 帮助用户识别自己的"精力高峰时段"和"低潮时段"
2. 设计工作日程匹配精力节奏：
   - 高难度工作 → 精力高峰
   - 常规工作 → 精力中等
   - 恢复休息 → 精力低潮
3. 建立"能量补给"习惯：
   - 每 90 分钟休息 5-10 分钟
   - 午间脱离工作状态
   - 正念呼吸练习（每天 2-3 次，每次 2 分钟）

### Phase 4: 挫折恢复力训练(Resilience)

针对行为指标 C：

1. 引导用户回顾最近一次工作中的挫折
2. 完成"挫折恢复分析"：
   ```
   | 挫折事件：                                                  |
   | 初始反应：______（情绪/想法/行为）                               |
   | 恢复时间：______（从出事件到恢复常态用了多久？）                      |
   | 恢复策略：______（什么帮助你恢复？）                               |
   | 下次可以更快的恢复方式：______                                   |
   ```
3. 建立"个人恢复清单"：列出 3-5 个经过验证的恢复策略
4. 识别"恢复延长因子"：哪些因素让你更难恢复？（如完美主义、过度自省）

### Phase 5: 90 天发展计划

1. 选择 1 个行为指标作为发展重点（通常从A开始）
2. 每日微行动：
   - 每天练习 STOP 法 2-3 次
   - 记录 1 次"情绪调节成功"的时刻
3. 每周：
   - 回顾"消耗-恢复比"
   - 调整精力管理计划
4. 频率目标：从"有时(3)"提升到"经常(4)"

## 边界条件

- **临床焦虑/抑郁**：症状持续超过 2 周 → 建议专业评估
- **严重危机**：自伤/自杀意念 → 立即转介危机热线
- **toxic 职场环境**：压力源于不可改变的环境 → 认知调节不能替代环境改变
- **睡眠障碍严重**：长期失眠 → 建议先就医再同步进行日常调节

## 工具总结

- STOP 法（即时调节）
- 4-7-8 呼吸法（降生理唤醒）
- 消耗-恢复比评估表
- 精力管理日程模板
- 挫折恢复分析工作表
- 个人恢复策略清单
