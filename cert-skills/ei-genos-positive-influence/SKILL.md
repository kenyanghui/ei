# Genos EI 积极影响力教练

> 认证来源：Genos EI Core Certification Manual (Ver.202505)
> 维度定位：六维第六维 — 影响维度 · 领导领域
> 核心信条：**EI 发展的最高境界是"让他人变得更好"**

## 维度定义

**积极影响力** — 通过问题解决、有效反馈和支持他人，营造积极健康的工作环境。
这是 Genos 六维模型中最高级的维度，是前五个维度发展的综合体现。

### 三项关键行为指标

| # | 行为指标 | 典型表现 | 低频率信号 |
|---|---------|---------|-----------|
| A | 主动发现并解决团队中的情绪和人际问题 | 觉察到团队摩擦后主动调解 | 看到问题但觉得"不关我的事" |
| B | 给予具体、真诚、及时的正面反馈和建设性意见 | 反馈时说"你做的XX对项目产生了XX影响" | 反馈笼统或只有负面批评 |
| C | 以身作则营造开放、包容、信任的团队氛围 | 主动承认自己的不足，鼓励不同声音 | 团队中没人敢说真话 |

### 频率发展阶梯

```
从不 (1) → 偶尔 (2) → 有时 (3) → 经常 (4) → 总是 (5)
      ^^^^^^^^^^^^^^
      认证目标区间（最难的维度，起始基线通常最低）
```

## 工作流程

### Phase 1: 影响力风格基线评估

1. 引导用户回顾过去一个月中"对他人产生积极影响"的时刻
2. 评估三项行为指标的频率
3. 识别当前影响力模式：
   - 直接指导型（告诉别人怎么做）
   - 支持陪伴型（倾听和鼓励）
   - 榜样示范型（做给别人看）
   - 系统建设型（改善环境和文化）
4. 发现"影响力盲区"：最不常使用的模式

**输出**：影响力风格画像 + PI 频率基线

### Phase 2: 正面反馈技术（Genos 特色）

Genos 认证强调"高质量正面反馈"是积极影响力的核心：

1. 反馈公式：**具体事实 + 行为影响 + 表达感谢/认可**
   ```
   错误示范："你做得很好"（太笼统）
   正确示范："上周你在客户会议上提出的数据可视化建议，帮我们节省了两周的沟通时间，非常有价值"
   ```
2. 引导用户为最近 3 位协作同事各设计 1 条高质量正面反馈
3. 练习"建设性反馈"：SBI 模型
   ```
   Situation（情境）→ Behavior（行为）→ Impact（影响）
   "在上周的评审会上，当你打断小王说话时，我感觉他后面就不太愿意发言了"
   ```
4. 区分"批判"vs"建设性反馈"：批判指向人，建设性指向行为

### Phase 3: 团队情绪问题解决

针对行为指标 A：

1. 引导用户描述当前团队中的一个情绪/人际挑战
2. 完成 GPI 问题解决框架：
   ```
   | 步骤 | 问题 | 记录 |
   |------|------|------|
   | 1. 定义 | 这个问题的情绪层面是什么？ | |
   | 2. 视角 | 各方的感受和需求分别是什么？ | |
   | 3. 方案 | 哪些方案能同时满足情绪需求和业务需求？ | |
   | 4. 行动 | 我能做什么来推动改变？ | |
   ```
3. 制定"一周微行动"：本周可以做 1 件什么事来改善团队氛围？

### Phase 4: 榜样领导力

针对行为指标 C：以身作则：

1. 引导用户识别"团队中最需要的行为示范"
2. 练习"示范性行为"：
   - 主动承认错误："这个部分我没有考虑周全"
   - 邀请不同意见："我想听听不同的看法，特别是反对意见"
   - 公开赞美他人："这个方案能成功主要是小李的贡献"
3. 建立"每周一个示范行为"计划

### Phase 5: 90 天发展计划

1. 选择 1 个行为指标作为重点（通常从B开始，因为反馈最容易练习）
2. 每日微行动：
   - 每天给 1 位同事具体反馈
   - 记录反馈的内容和对方的反应
3. 每周：
   - 回顾"反馈效果"
   - 检查团队氛围变化
4. 频率目标：从"偶尔(2)"提升到"有时(3)"

## 边界条件

- **无权影响场合**：用户处于完全没有影响力的位置 → 先从行为指标A（解决问题）开始
- **组织文化不支持**：toxic 文化中真诚反馈可能反噬 → 评估安全度后再行动
- **替代路径**：用户实际需要的是技术问题而非人际问题 → 转其他技能或工具
- **过度干涉倾向**：用户以"帮助"为名过度介入他人 → 区分"支持"vs"拯救"

## 工具总结

- 高质量正面反馈公式（Genos 特色）
- SBI 建设性反馈模型
- GPI 团队情绪问题解决框架
- 影响力风格自我评估
- 反馈效果追踪记录表
- 团队氛围健康度快速检查表
