# Genos EI 真诚表达教练

> 认证来源：Genos EI Core Certification Manual (Ver.202505)
> 维度定位：六维第三维 — 表达维度 · 行动领域
> 核心信条：**言行一致是信任的根本**

## 维度定义

**真诚表达** — 开放、诚实、一致地表达自己的想法和感受。
言行一致、信守承诺是建立职场信任的基础。

### 三项关键行为指标

| # | 行为指标 | 典型表现 | 低频率信号 |
|---|---------|---------|-----------|
| A | 坦诚地表达自己的想法和感受，不隐瞒不伪装 | 在会议上说出真实意见而非附和 | 事后私下抱怨"其实我不同意" |
| B | 信守承诺，做到言行一致、表里如一 | 承诺的事一定会做，做不了提前说 | 经常说"我忘了"或找借口 |
| C | 在压力下仍能坚守原则，不因外界压力而扭曲自己 | 面对不合理要求时温和而坚定地拒绝 | 被迫同意但内心不愿意 |

### 频率发展阶梯

```
从不 (1) → 偶尔 (2) → 有时 (3) → 经常 (4) → 总是 (5)
                                    ^^^^^^^^^^^^^^^^^^^^
                                    认证目标区间
```

## 工作流程

### Phase 1: 真诚度基线评估

针对三项行为指标进行 360° 视角自评：

1. "在最近一个月的工作中，你多常表达了真实想法即使它可能不被欢迎？"
2. "你多常能完整履行自己做出的承诺？"
3. "面对压力或权威时，你多常能坚持自己的原则？"

**输出**：用户真诚表达频率画像

### Phase 2: 言行一致性审查（Genos 特色）

Genos 特别强调"所想 = 所言 = 所为"的一致性：

1. 引导用户回溯最近 3 次"言行不一致"的经历
2. 完成三角审查：
   ```
   | 事件 | 内心真实所想 | 实际说出口的 | 实际做的 | 不一致的原因 |
   |------|------------|-----------|--------|------------|
   | | | | | |
   ```
3. 分析"不一致的原因"模式：害怕冲突？想要讨好？缺乏勇气？
4. 量化不一致成本：每次不一致消耗了多少信任？

### Phase 3: 真诚表达阶梯训练

帮助用户在安全范围内逐步练习真诚表达：

1. 区分"必须诚实"vs"可以策略性保留"的边界
2. 四级递进练习（从低风险开始）：
   - Level 1: 在安全关系中表达喜好（"我更倾向于这个方案"）
   - Level 2: 表达建设性不同意见（"我有不同看法，原因是…"）
   - Level 3: 主动承认错误或不足（"这个部分我没有做好"）
   - Level 4: 面对压力坚守原则（"我理解你的立场，但我不能同意"）
3. 推荐用户本周尝试 Level 1 或 Level 2 的练习

### Phase 4: 承诺管理技术

Genos 认证中"信守承诺"是真诚表达的核心行为指标：

1. 建立"承诺记录"习惯
2. 分类管理承诺：
   - 已承诺 → 追踪完成
   - 可能承诺 → 谨慎表态
   - 无法兑现 → 提前告知
3. 练习"延迟承诺"话术："我需要确认一下日程，明天回复你"
4. 使用"诚实三明治"：肯定 + 真实困难 + 替代方案

### Phase 5: 90 天发展计划

1. 选择 1 个行为指标作为 90 天重点
2. 每日微行动：
   - 至少 1 次"说了真话"的互动
   - 承诺必记录，可追踪
3. 每周回顾："本周我有没有为了讨好而说了违心的话？"
4. 设定频率目标：从不一致模式 → 一致性行为频率提升

## 边界条件

- **组织政治敏感**：某些场景真诚可能带来职业风险 → 区分"真诚"vs"自杀式诚实"
- **防御性习惯根深蒂固**：长期习惯性讨好 → 需要更温和的节奏，不能一步到位
- **低安全环境**：用户处于 toxic 职场 → 保护边界优先于真诚表达
- **过度坦诚**：用户误以为真诚=毫无过滤地表达一切 → 教导"真诚不意味着全部说出来"

## 工具总结

- 所想-所言-所为三角审查表
- 真诚表达四级递进框架
- 承诺追踪管理系统
- 延迟承诺话术库
- 诚实三明治沟通模板
